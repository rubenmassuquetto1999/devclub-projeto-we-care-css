# 🐾 We Care - About Your Pet

Uma landing page responsiva e moderna para serviços de cuidados com pets. Este projeto foi desenvolvido para treinar a estruturação semântica de HTML e técnicas avançadas de CSS, como Flexbox e Media Queries.

O design original foi criado no Figma e traduzido de forma fiel para o código, garantindo uma excelente experiência tanto em monitores grandes quanto em dispositivos móveis.

## 🚀 Funcionalidades

* **Totalmente Responsivo:** Layout adaptável para Desktops, Tablets e Smartphones (iOS e Android).
* **Mobile-First Design:** Adaptação limpa para celulares, escondendo a barra de navegação e focando na conversão (botão "Explore"), conforme o design original.
* **Layout Fluido:** Uso intensivo de CSS Flexbox para alinhamento e distribuição de elementos.
* **Tipografia Personalizada:** Integração com o Google Fonts utilizando a família tipográfica *Montserrat*.
* **Interatividade:** Efeitos de transição suaves (hover) em botões e links para melhor experiência do usuário (UX).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica (`<header>`, `<nav>`, `<main>`, `<section>`).
* **CSS3:** Estilização, Flexbox, Media Queries (`@media`) e Transições.

## 💻 Como rodar o projeto

Este projeto não possui dependências complexas ou necessidade de servidores locais. Para visualizá-lo:

1. Faça o download ou clone este repositório.
2. Extraia os arquivos na sua máquina.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador padrão.
